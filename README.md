# Server-Performance-Stats

A shell script that analyzes basic server performance stats on any Linux server.

## How to Run

1. Clone the repo:
git clone https://github.com/loyditech/Server-Performance-Stats.git

2. Go into the folder:
   cd Server-Performance-Stats

3. Make the script executable:
   chmod +x server-stats.sh

4.Run it:
./server-stats.sh

5. For failed login stats, run with sudo:
   sudo ./server-stats.sh

   ## What it shows
- Total CPU usage
- Total memory usage (Free vs Used including percentage)
- Total disk usage (Free vs Used including percentage)
- Top 5 processes by CPU usage
- Top 5 processes by memory usage
- OS info, uptime, load average
- Logged in users
- Failed login attempts
